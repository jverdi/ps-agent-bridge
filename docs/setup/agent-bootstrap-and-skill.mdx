Use this page when you want a fast local setup for:

- CLI (`psagent`)
- MCP server (`psagent mcp-serve`)
- Photoshop UXP bridge plugin reload flow
- Codex skill (`psagent-cli`)

## One-time shell bootstrap (copy/paste)

Run from repo root:

```bash
npm install && npm run build
npm run skill:install
npm link
```

This installs dependencies, builds the CLI, installs the local skill to `$CODEX_HOME/skills/psagent-cli`, and links `psagent` globally.

## Start UXP + daemon + MCP

Terminal A:

```bash
psagent bridge daemon
```

Terminal B:

```bash
npm run bridge:reload
```

Terminal C (when needed for agent tool use):

```bash
psagent mcp-serve
```

## One prompt humans can paste into an agent

Copy/paste this prompt into Codex (or another coding agent):

```text
Set up PSAgent Bridge in this repo end-to-end and verify it works.

Requirements:
1) Install/build CLI and expose `psagent` command.
2) Install the local skill from `skills/psagent-cli` into `$CODEX_HOME/skills/psagent-cli`.
3) Start or verify daemon on `http://127.0.0.1:43120`.
4) Reload the Photoshop UXP plugin with `npm run bridge:reload`.
5) Confirm bridge status is connected.
6) Start MCP (`psagent mcp-serve`) and report command + expected tools.
7) Run one smoke ops payload that creates text `success` and then cleans up.

Return a compact checklist with pass/fail per step and exact commands used.
```

## Optional GitHub-based skill install

If you want to install the skill directly from GitHub using the Codex system installer script:

```bash
python3 ~/.codex/skills/.system/skill-installer/scripts/install-skill-from-github.py \
  --repo jverdi/cccli \
  --path skills/psagent-cli
```

Then restart Codex so it loads the newly installed skill.
